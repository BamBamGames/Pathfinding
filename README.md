# Pathfinding
Many pathfinding algorithm



## 包含的算法

* Basic
  * A*
  * BFS
  * Dijkstra
* Bidirectional Search
  * Bidirection A*
* Any-Angle Search
  * Theta*
    * Lazy Theta*
* Incremental Search
  * D*
  * LPA*
  * D* Lite
  * Path Adaptive A*
* Moving Target
  * GAA*
  * G-FRA*
* Other Search
  * Flow Field
  * Jump Point Search
    - JPS Plus



## 怎么玩

### 选择算法

运行Pathfinding场景，选中Search这个物体，然后其Inspector面板中选择想要的寻路算法，按空格键来运行算法

![](F:/Kaima/Pathfinding/Doc/Search_go.png)

### 网格操作

- 按住可以拖动起点和终点的位置
- 鼠标左键添加阻挡
- 鼠标右键移除阻挡
